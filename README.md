## ğŸ¨ Customization

### Colors
The website uses a warm, food-themed color palette. Update the `:root` section in `css/style.css`:

```css
:root {
    --primary-color: #ff6b35;    /* Orange - appetite stimulating */
    --secondary-color: #f7931e;  /* Warm orange */
    --accent-color: #4caf50;     /* Fresh green */
    /* ... more variables */
}
```

### Content
1. **Menu Items**: Update food descriptions and prices in the menu section
2. **Product Images**: Add your food photos to the `images/` folder
3. **Restaurant/Store Info**: Update location, hours, and contact details
4. **Categories**: Modify product categories to match your offerings

### Sections
The website includes these main sections:
- **Hero Section**: Eye-catching landing area with food-themed animations
- **About Section**: Company story and quality commitment with statistics
- **Products Section**: Food categories with detailed descriptions
- **Menu Section**: Featured dishes and signature items
- **Contact Section**: Order form and restaurant/store information
- **Footer**: Additional links, categories, and contact details
